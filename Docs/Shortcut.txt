*) Default Shortcut key:
    -> Press `CTRL+Shift+P` OR `F1` (open command palette)
    -> 'ctrl + p' (go to file)
    -> 'ctrl + p & ctrl + g' (go to particular line)
    -> 'ctrl + f4' (to close editor file)
    -> 'shift + f10' (to click mouse right click)
    -> 'shift + up' (to zoom in)
    -> 'shift + down' (to zoom out)
    -> 'ctrl + tab' (switch between opened file)
    -> 'ctrl + shift + l' (to select all similar selected word/line/phrase/code)
    -> 'ctrl + /' (toggle comment)
    -> 'ctrl + shift + k' (to delete line)
    -> 'ctrl + f' (open search bar)
    -> (hover on function/variable/keyword) + 'f12' (show defination of that)

    *) Terminal
        -> search for `Terminal: select default profile`
        -> select the terminal as your choice to make it a default terminal
        -> 'ctrl + Shift + `' (to open new terminal)\
        -> `ctrl + Shift + C' (open exter on same directory)
        -> 'ctrl + pageup' (to open above terminal)
        -> 'ctrl + pagedown' (to open below terminal)
        -> 'ctrl + alt + pageup' (to scroll up terminal)
        -> 'ctrl + alt + pagedown' (to scroll down terminal)
    *) SideBar
        -> 'ctrl + 0' (focus sidebar)
        -> 'ctrl + b' (open and close sidebar)
        -> 'ctrl + shift + e' (switch to file explorer sidebar)
        -> 'ctrl + shift + f' (switch to search sidebar)
        -> 'ctrl + shift + g' (switch to source control sidebar)
        -> 'ctrl + shift + d' (switch to debug sidebar)
        -> 'ctrl + shift + x' (switch to extension sidebar)
